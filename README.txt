Purpose: A campgrounds website based on the YelpCamp website created in Colt Steele's web developer bootcamp on Udemy. Created using Node.js.

YelpCamp allows a user to: 
- register 
- login 
- create new campgrounds
- edit campgrounds 
- delete campgrounds
- explore different campgrounds created by other users and themself  

How run:
    - install dependencies
    - run command 'npm start'
